Multi-Proyecto API

API para la gestión de múltiples proyectos y usuarios.

Requisitos

  • Node.js (v14 o superior)
  • MongoDB
  • npm o yarn

Instalación

  1. Clonar el repositorio
  2. Instalar dependencias: npm install
  3. Crear archivo .env basado en config/.env.example
  4. Configurar las variables de entorno necesarias

Uso

Desarrollo

npm run dev

Producción

npm start

Poblar base de datos

npm run seed

Estructura del Proyecto

multi-proyecto-api/
├── src/
│   ├── api/           # Rutas y controladores
│   ├── models/        # Modelos de MongoDB
│   └── utils/         # Utilidades y middleware
├── public/            # Archivos estáticos
├── config/            # Archivos de configuración
├── scripts/           # Scripts de utilidad
├── .env.example       # Ejemplo de variables de entorno
├── package.json       # Dependencias y scripts
└── README.html        # Documentación

Variables de Entorno

Crear un archivo .env en la raíz del proyecto con las siguientes variables:

PORT=5000
MONGO_URI=mongodb://localhost:27017/multiproyecto
JWT_SECRET=tu_clave_secreta_aqui

Endpoints

Autenticación

  • POST /api/auth/register - Registrar nuevo usuario
  • POST /api/auth/login - Iniciar sesión

Usuarios

  • GET /api/users - Obtener todos los usuarios
  • GET /api/users/:id - Obtener usuario por ID

Proyectos

  • GET /api/projects - Obtener todos los proyectos
  • POST /api/projects - Crear nuevo proyecto
  • GET /api/projects/:id - Obtener proyecto por ID
